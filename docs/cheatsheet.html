📋 NoorCast Cheat Sheet

Every feature at a glance. canvas = visible in recording. HTML = teacher-only. Pro = Pro mode only.

⌨ Keyboard Shortcuts

R
Record / Stop
Start or stop recording. Countdown plays first if enabled.
L
Laser pointer
Red dot follows your cursor on the canvas. Visible in recording.
F
Freeze screen
Pauses the canvas at the current frame. Combine with Draw to annotate.
D
Draw / Whiteboard
Freehand drawing on the canvas. Pick color and thickness from the palette.
S
Snapshot
Captures the current canvas as a PNG image.
T
Teleprompter
Auto-scrolling script overlay. Teacher-only, never in recording.
Z
Zoom
Click a point on canvas to zoom in 2x. Click again to reset.
M
Chapter marker
Drops a timestamp marker. Exports as .vtt chapter file for YouTube.
Q
Quiz card
Shows a quiz/question overlay on canvas during recording.
G
Grid overlay
Rule-of-thirds grid for framing. Teacher-only, not in recording.
1-6
Switch scene
Instantly switch to scene 1-6. Each switch drops a chapter marker.
?
Random scene
Jump to a random scene. Fun for quizzes and surprises.
Ctrl+Z
Undo
Undo last source position/size change.
Ctrl+Y
Redo
Redo undone layout change.
Ctrl+S
Download all
Downloads video + chapters + sensors in one click.
⇧F
Focus mode
Hides all panels — canvas only. Press again to restore.
Alt+1-9
Toggle source
Show/hide source #N without removing it.
Esc
Stop / Exit / Close
Stop recording, close modals, close any open Settings/Help/Log panel, deselect sources, exit modes.
Ctrl+K
Global search (Spotlight)
Universal command palette. Searches scenes, skins, backgrounds, themes, mascots, settings, tools, sounds, badges, hotkeys.
/
Global search (alt)
Same as Ctrl+K. Single-key shortcut from anywhere.
C
Live captions
Toggle browser SpeechRecognition captions. Exports as .srt.
P
Pause / Resume
Pause without stopping. Press again to resume. Auto-pause on tab switch (if enabled).
Ctrl+A
Select all sources
Multi-selects every visible source. Drag any to move all together.
Shift+click
Add/remove from selection
Toggles a source in/out of multi-select. Sky-blue dashed outline.
Alt+drag
Snap to 48px grid
Hold Alt while dragging — source snaps to grid + visible guides appear.
Ctrl+wheel
Smooth zoom toward cursor
On the stage, Ctrl+scroll = continuous zoom (1× to 4×) centered on cursor.
Ctrl+Shift+D
Debug HUD
Toggle FPS + JS heap memory overlay (top-right).
Delete
Remove selected source
Deletes the selected source(s). Backspace also works. Streams stopped.
