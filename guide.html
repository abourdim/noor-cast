Getting Started

What is NoorCast?

A free, browser-based multi-camera screen recorder for teachers creating tutorials with robots and micro:bits. No install, no account, no cloud. 100% local.

Requirements: Chrome or Edge (desktop). Works offline after first load (PWA).

Simple vs Pro Mode

Simple Mode

Hides advanced tools. Right sidebar collapsed. Perfect for beginners.

Pro Mode

All tools visible. Scenes sidebar open. Full control.

Switch anytime in Settings.

First Steps

  1. Add a Screen source (blue button)
  2. Add a Camera (dropdown + green button)
  3. Select a Mic
  4. Click REC
  5. Click Stop and download

micro:bit

Connect

Green Connect micro:bit button on canvas (draggable). Web Bluetooth (Chrome/Edge only).

Controls

D-pad

CMD:LEFT/RIGHT/UP/DOWN/CLEAR commands.

Servos

Pan + Tilt (0-180). Synced with canvas gauges.

LED Grid

5x5 toggleable. Mirrors micro:bit LEDs.

Joystick

CMD mode (blue) or Servo mode (green).

Pilot Scene

Scene 6 shows D-pad + servos + joystick + LEDs as draggable overlay on canvas.

Sensor Overlay

Accelerometer, temp, light, sound, compass on canvas. Toggle: More → Fun → Sensors.

Robot Choreography

Record UART commands with timestamps. Replay in sync. Persisted in localStorage.

Fun Features

Access: toolbar → More → Fun section.

AI Co-host

36 reactive SVG characters (Science, Space, Hacker, Pirate, Cyber). Reacts to voice: claps on pause, prompts scene changes, shows time cards.

Canvas Pets

12 animated walkers: Robot, Anonymous, Skull, Agent, Bug, UFO, Drone, Arc, Spider, Eye, Pencil, Wrench. Freeze when loud. Multiple active. Baked in.

Voice Commands

"record", "stop", "screenshot", "zoom", "laser", "fire", "scene 3". FR/EN/AR.

Smart Scene

Auto-switches scenes on keywords ("code" → Code scene, "robot" → Robot scene).

XP & Challenges

1 XP/sec recording. Level up every 300 XP. Daily challenge = +25 XP. Unlock skins at XP milestones.

Combos

3+ features = COMBO, 5+ = SUPER, 7+ = ULTRA. Bonus XP.
