# NoorCast → Etsy 1-Minute Demo Playbook

**For Etsy listing videos (60-second cap, 9:16 portrait, autoplay-muted preview).**

A second-by-second script: **what to say** + **what to click in NoorCast** to produce a tight Etsy demo that converts. Keep this open on a second monitor or print A4.

---

## Etsy video specs (memorise once)

| Spec | Value |
|---|---|
| **Length** | Max 60 seconds (90s for some listings; aim for 55s to be safe) |
| **Aspect** | **9:16 portrait** (mobile-first; Etsy autoplays in feed) |
| **Resolution** | 1080×1920 minimum |
| **Codec** | MP4 / H.264 + AAC audio |
| **File size** | Under 100 MB |
| **Audio** | Plays muted by default in feed → **captions are mandatory** |
| **Cover frame** | First frame becomes the thumbnail → make it count |
| **Hook window** | First 3 seconds determine if viewer keeps watching |

---

## Pre-flight checklist (90 seconds, do once per shoot)

Open NoorCast and tick these BEFORE you press REC:

- [ ] Settings → 📐 **Stage format → 9:16 (1080×1920)** *(Reels mode auto-on)*
- [ ] Toolbar → **📱 Reels** button is highlighted *(safe-zone guide visible)*
- [ ] Drag **Watermark** + **Sensors overlay** + any branding INSIDE the green dashed box
- [ ] Add at least one source — for Etsy demos: **🎥 Camera** (face) + **🖥 Screen** OR **🎥 Camera** alone
- [ ] If face-cam: turn ON **🎯 Auto-frame** (source toolbar → Smart frame) → keeps you centered when you move
- [ ] Settings → 💬 **Live captions ON** + **🔥 Burn into video** *(Etsy auto-mutes — captions = your voice)*
- [ ] Settings → 🎥 Capture → **🎙 Skip silent intro ON** *(skips dead air at the top)*
- [ ] Settings → Recording quality → **High (8 Mbps)** *(default — verify)*
- [ ] Header status bar should read: `🟢 Ready · 2 sources · 9:16 📱 Reels · 8 Mbps`
- [ ] Mode badge: 🧒 Kids if a child is on camera, 🧑‍🏫 Teacher otherwise
- [ ] Quiet room. Phone face-down. Kids/dogs warned.
- [ ] **Practice run once** without recording — say the script out loud, time it.

## The 60-second arc (universal structure)

```
 0–3 s   HOOK         "Stop scrolling — here's what this does in 60 seconds."
 3–10 s  CONTEXT      Show the problem. Big visual.
10–35 s  DEMO         Do the thing. Switch scenes once or twice for energy.
35–50 s  PAYOFF       Show the result. Smile. Pause for 1 second.
50–55 s  CTA          "Link in description / shop link below."
55–60 s  END FRAME    Logo + "@yourhandle" frozen for the loop preview.
```

Etsy feed loops the video, so the **end frame matches the cover** — keeps viewers watching the loop.

## Script template A — "I built this thing" (creator showcase)

**Use case:** showing off what you make/sell. Etsy's most common video format.

| Time | What to SAY | What to CLICK in NoorCast |
|---|---|---|
| 0–3 s | *"Three seconds — here's a [PRODUCT]."* | (Already recording. Big smile, hold the product up if physical.) |
| 3–6 s | *"Most people don't know it can [DO COOL THING]."* | If demo is on-screen: **toolbar → Scene 💻 Code** (full-screen the screen source) |
| 6–15 s | *"Watch this — I'll show you in one move."* | Do the action. **Press M** to drop a chapter marker at the climax |
| 15–35 s | *"Here's how it works behind the scenes…"* | **Scene 🎛 Sensors** (split: cam + sensors) if relevant. **🔴 Laser tool (L)** to point at things on screen |
| 35–45 s | *"And the result is…"* | **Scene 👋 You** (face-cam fullscreen). Smile. Hold for 2 s. |
| 45–55 s | *"You can grab one in my Etsy shop — link below."* | (Stay on face-cam. Eye contact with lens.) |
| 55–60 s | *"Thanks for watching."* | (Stay still 2 s — this becomes the loop frame.) |

**Use case:** instructional. Tells the buyer what they're getting.

| Time | What to SAY | What to CLICK in NoorCast |
|---|---|---|
| 0–3 s | *"60 seconds to learn [SKILL]."* | **🔴 Laser** ON. Point at the most interesting on-screen element. |

**Use case:** showing a micro:bit / robot project — kid-targeted Etsy listing.

| Time | What to SAY | What to CLICK in NoorCast |

## During-recording tactics

- **Press `M`** every time you do something interesting → chapter markers. After STOP, click **✨ Auto-chapter from transcript** in the SHARE column to auto-add more.
- **Press `L`** for laser pointer when explaining the screen.
- **Press `F`** to freeze the canvas for 1 second when emphasizing a result.
- **Switch scenes only on words**, never mid-sentence — feels disorienting otherwise.
- **Look at the camera**, not the screen, during all face-cam moments. Etsy buyers buy from people who look them in the eye.
- **Smile in the last 5 seconds**. The loop frame matters.
- **Watch the `🔴 Recording` status bar** — if it turns red, you've crossed 90 s and Etsy may auto-trim.

## Common mistakes to avoid

| ❌ Don't | ✅ Do |
|---|---|
| Start with "Hi I'm…" + 10s intro | Start with the result/hook in the first 3 seconds |
| Talk over busy on-screen action | Pause, let the visual breathe, then explain |
| Switch scenes every 2 seconds | One scene change per phase of the script |
| Forget to test audio | Settings → 🎤 → speak → check the level meter |
| Record with messy desk | Clean the area in camera frame — even a small frame counts |
| Use a Pro-mode toolbar in a kid-targeted video | Switch to 🧒 Kids mode — chunkier, friendlier vibe in the cuts |
| End mid-sentence | Land your closing line, then hold 2 seconds of silence |

## After you press STOP

1. Take modal opens automatically. Watch the playback **once**, all the way through. Eyes off your script — does it land?
2. If your take is over 60 s → click **📱 Cut to 90 s for Reels** (the in-browser trim re-encodes; for finer trims use ffmpeg).
3. Click **✨ Auto-chapter from transcript** → adds smart chapter markers based on your speech (great for the SRT export).
4. Note the **MISSION COMPLETE card** — your "cool factor" gives you a vibe-check before publishing.
5. Click the title field, type a clean filename like `etsy-listing-v1.mp4`.
6. Click the green **Download** button → save MP4 to a folder.
7. (Optional) Click **💬 Captions (.srt)** → save the subtitles separately. Etsy doesn't accept SRT directly but you can use it to write the listing description.

## The 5-take rule

**Never publish your first take.** The script feels different out loud than it reads. Plan to do 3-5 takes:

1. **Take 1** — read the script literally. Find the awkward bits.
2. **Take 2** — fix the awkward bits, less reading more talking.
3. **Take 3** — drop the script entirely. Talk to a friend, not to a camera.
4. **Take 4** — slow down 10 %. Add 1 deliberate pause.
5. **Take 5** — your keeper.

Each take auto-saves. Use the **🎬 Last take** header button to revisit any of them.

## Etsy-specific upload tips

- Etsy listing videos appear at the top of the listing as a **autoplaying preview tile**. The **first frame is your cover** — make it visually clean (face, product, logo, no half-words).
- Etsy compresses uploads — render at 8 Mbps minimum so the compressed version still looks crisp.
- Use the listing **title** to match the video's first words. *"60 seconds to learn macramé"* video → listing title starts *"Learn macramé in 60 seconds — full kit"*.
- Add the captions text to the listing **description** — searchable, helps SEO.
- Tag the listing with the words you actually said in the video.
