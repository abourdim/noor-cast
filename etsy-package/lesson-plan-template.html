    Sample Lesson: My First Micro:bit Tutorial

    Example lesson plan using NoorCast
    Making a Digital Pet with Micro:bit
    45 minutes
    Ages 8-12
    STEM / Coding / Robotics
    Learning Objectives
    1. Students will connect a micro:bit to NoorCast via Bluetooth
    2. Students will create a simple LED animation program
    3. Students will record a tutorial explaining their code

    NoorCast Features Used in This Lesson

    • Pilot Scene micro:bit overlay with D-pad, LED grid, servo gauges
    • Freeze pause the screen to annotate with the laser
    • Markers create chapters for easy navigation
    • Sensor Overlay show accelerometer/temperature data live on canvas
    • Auto Thumbnail generate a cover image from the best frame
    • Quality Score check audio levels, duration, scene variety
